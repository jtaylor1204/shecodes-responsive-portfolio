Jammmin: Spotify Playlist Generator

This app features a modern, responsive design, utilizing React JS and the Spotify developer API to allow users to create playlists that will be saved to their Spotify account. Jammmin also features an intuitive, user-friendly interface, allowing users to easily navigate the app to create their playlists!

Ravenous: Yelp Food Search App

This React JS project focuses on fetching data from the RESTful Yelp API, and allows users to search for restaurants in an area of their choice. Users will enter a restaurant name or cuisine type and will return results with a list of restaurants matching the search query, ordered by highest rated, most reviewed, and best match. Each restaurant listing includes basic information such as ratings, location, and address.
